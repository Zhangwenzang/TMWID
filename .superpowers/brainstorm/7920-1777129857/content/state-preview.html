核心交互概念

气泡行为定义

🔵 悬浮位置：桌面边缘或屏幕角落，半透明，不遮挡内容
🔄 状态切换：默认隐藏 → 有任务时显示 Working → 完成时切换 Work Done
⚡ 动效触发：状态切换时播放对应 MP4，播完回到安静状态
👆 用户交互：点击可关闭/最小化，再次唤醒通过菜单或快捷键
